# CharacterAICord

Connect a characterAI bot to your discord account as a Self-bot, made with discord.js-selfbot-v13 &amp; node_characterai

# bot.js

Was tested on these versions months ago. Hence why this is archived.

  "discord.js-selfbot-v13": "^3.7.0",
        "node_characterai": "^2.0.0-beta.12",

It was tested and worked on these, I don't know if c.ai changed stuff after that. You might get problems related to cloudflare now.

Replace the characterId, conversationId, Discord token and C.ai token variables with your own.

Guide on how to get those: https://github.com/realcoloride/node_characterai
